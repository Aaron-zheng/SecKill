# SecKill
秒杀系统的搭建和记录 SSM (Spring+SpringMVC+Mybatis)

## 开发环境
1. JRE: 1.8.0_40-release-b132 x86_64
2. JVM: OpenJDK 64-Bit Server VM by JetBrains s.r.o
3. IntelliJ IDEA 2016.1
4. MacOS 10.12.3
5. MySQL Distrib 5.7.17


## 配置
1. junit 4.11
2. mysql 5.1.35
3. mybatis 3.3.0
4. spring 4.1.7
